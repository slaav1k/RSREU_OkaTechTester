# RSREU_OkaTechTester

Этот репозиторий содержит автоматизированные тесты для проекта [ОкаТех - Магазин электротехники](https://github.com/slaav1k/springLabs).

## Что использовалось?

Тесты написаны с использованием **Cucumber** (BDD-фреймворк для поведения-ориентированного тестирования) и **Selenium WebDriver** (для автоматизации браузера). Они покрывают ключевые сценарии пользовательского взаимодействия.

Тесты выполнялись на **Java 21** с использованием **Maven** для сборки и управления зависимостями. WebDriver настроен для **Firefox** (с возможностью переключения на Chrome).
